# App flask para gerenciamento de varal inteligente

## O que ele faz?
O aplicativo vai ser capaz de controlar um varal inteligente (ESP32) feito no TCC da minha namorada

## Dependências
- [flask]
- [python3]

## Como iniciar?
```sh
git clone https://github.com/amendoa657/AppFlaskManageVaral.git
cd AppFlaskManageVaral
python -m venv venv
pip install flask
python app.py
```
